Title: Welcome to C3 AI Data Integration Starter Package

Welcome to the C3 AI Data Integration starter package! 

## Setup steps

1. Follow steps in the <a href="https://learn.c3.ai/path/data-integration/hands-on-build-a-data-integration-pipeline/321241/scorm/243ucjhbtic73" target="_blank">Get Started</a> lesson for instructions on deploying the starter package.
2. This starter package is designed to accompany the <a href="https://learn.c3.ai/path/data-integration/hands-on-build-a-data-integration-pipeline" target="_blank">Build a Data Integration Pipeline</a> course. It will guide you through building a data pipeline from a file source step by step.


## DI Pipeline overview

The goal of this course is to guide students through the process of loading data into an application where both the Entity and Canonical Types are already defined. The objective is to map the data into the appropriate Canonical Type—after which the application handles the remaining processing automatically.
